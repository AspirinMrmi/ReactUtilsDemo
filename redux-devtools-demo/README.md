## redux-devtools-demo 项目内容
简单介绍了 redux-devtools-extension 插件的基本用法。


----- 


## 项目运行

### 准备工作
项目下载完成之后，打开项目，打开命令行，运行以下命令:
``` bash
$ cd redux-devtools-demo
$ npm install
```



### 查看运行结果
在命令行运行：
``` bash
$ npm run server
```
即可在浏览器中看到输出结果。
